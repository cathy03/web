자주 묻는 질문

  • COMMON

    가입 시 서류 제출이 필요한가요?

    자격 인증에 이용된 정보는 조회만하고 저장하지않아서 안전합니다.
    [자산, 직장, 사업가] 조건에 해당하는 경우, 디지털 간편 인증으로 서류 제출없이 가입이 가능합니다.
    [전문직, 자산] 조건은 합격증 또는 면허번호, 자산 인증 서류 제출이 필요합니다. (해당 조건들도 간편인증으로 대체할 예정입니다.)

  • COMMON

    연령 제한이 있나요?

    만 19세 이상 성인, 만 43세 미만까지만 가입이 가능합니다.

  • COMMON

    가입 심사 중이라고 뜨는데 얼마나 기다려야하나요?

    서류 심사는 당일 내에 모두 처리하고 있습니다.
    최대한 빨리 심사하도록 최선의 노력을 다하겠습니다.

  • COMMON

    이미 가입했는데, 드라이버 또는 패신저로 서비스를 이용할 수 있나요?

    풀러팅은 계정 당 하나의 역할만 이용 가능합니다.
    별도의 계정으로 회원가입을 통해 패신저 모드 이용이 가능합니다.

  • DRIVER

    법인, 리스, 가족명의 차량인 경우, 등록 방법

    안내된 이메일 주소(helps@poolirting.com)로 증빙자료를 보내주시면 차량 등록이 가능합니다.
    1. 가족명의 차량 : 차량등록증, 가족관계증명서, 차량 사진
    2. 회사 차량 : 차량 등록증, 사업자등록증, 재직증명서, 차량 사진
    3. 리스차량(본인) : 차량 등록증, 리스 계약서, 차량 사진
    4. 리스차량(가족) : 차량등록증, 가족관계증명서, 리스계약서, 차량 사진
    5. 리스차량(법인대표) : 차량등록증, 사업자등록증, 리스계약서, 차량 사진
    모든 서류는 제출 본인을 기준으로 본인과 차량소유주의 관계가 증명이 되어야 합니다.
    모든 서류에 주민등록번호 뒷자리는 가리고 보내주세요.
